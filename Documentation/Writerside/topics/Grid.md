# Grid

Das Grid Layout ist ein zwei-dimensionales Layout-System. Mit Grid kann man Inhalt in Reihen und Spalten organisieren.

## display

Dieses Property definiert das Element als Grid-Container.

````CSS
.container {
  display: grid | inline-grid;
}
````
