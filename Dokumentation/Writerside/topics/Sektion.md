# Sektion

Das `<section>`-Element repräsentiert eine Sektion in einem Dokument, welche keine weitere spezifische semantische Elemente mehr hat, die es 
repräsentieren.

```HTML
<section>
  <h2>Heading</h2>
  <p>Bunch of awesome content</p>
</section>
```
